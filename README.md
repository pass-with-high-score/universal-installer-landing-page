App name: Universal Installer

Short description: Install APK, APKS, XAPK & APKM with split APK support and silent install.

Universal Installer is a modern Android package manager that handles what the default installer can't — split APKs, bundled packages, and silent installations.

<b>Supported Formats</b>
• APK — Standard Android packages
• APKS — Split APK bundles
• XAPK — Compressed split bundles (APKPure format)
• APKM — Split bundles (APKMirror format)

<b>Key Features</b>
• Install split APK packages that the default installer rejects
• Preview APK details before installing — app name, version, permissions, architectures, languages
• VirusTotal malware scanning — check files for threats before installation
• Installation history — track every install with status and timestamp
• Batch uninstall — long-press to select multiple apps and remove them at once
• Search and filter installed apps including system apps
• Auto-delete source file after successful install (optional)
• Light, dark, and system theme support with Material You

<b>Shizuku Mode (Optional)</b>
For advanced users with Shizuku installed:
• Silent install without user prompts
• Replace existing packages
• Allow version downgrades
• Auto-grant permissions
• Install for all users
• Bypass low target SDK restrictions
• Allow test/debug packages

<b>Privacy</b>
• No internet required for core functionality
• VirusTotal scanning is optional and requires your own API key
• No ads, no tracking, no data collection
• Fully open source — inspect the code yourself
